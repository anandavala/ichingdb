About

This is a prototype of a new "I Ching Engine". The front-end website is deliberately minimal, showing only the core functionality. This can be customised in the future.

The purpose of this website is to exhibit the functionality of the back-end engine, which is a simple MySQL database and Django to interface this with the web.

This work is unlicensed and free to use and adapt in any way you see fit.
